## How to run the project?

- Step One

  ```shell
    $ git clone https://github.com/KassioVieira/WeatherApp.git
  ```

- Step Two
  ```shell
   $ cd WeatherApp && yarn
  ```
- Step Three
  ```shell
   $ cd ios && pod install
  ```
- Step Four

  - Change the file name `.env.example` to `.env` and add your credentials.

- Step Five

  ```shell
  $ yarn ios
  ```

  or

  ```shell
    $ yarn android
  ```

# About this project
